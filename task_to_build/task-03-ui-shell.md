## Objective
Set up the application layout, navigation, and global styling based on shadcn/ui.

## Steps
- [ ] Configure Tailwind Theme <!-- id: 0 -->
    - Set primary color to Emerald-600 (`142.1 76.2% 36.3%`) as per PRD.
    - Ensure fonts are correctly applied.
- [ ] Create Layout Components <!-- id: 1 -->

    - `Footer.tsx`: Simple copyright/credits.
- [ ] Update Root Layout <!-- id: 2 -->

    - Ensure responsive container classes.
- [ ] Install Required shadcn Components <!-- id: 3 -->
    - `button`, `input`, `card`, `select`, `badge`, `skeleton`, `toast`, `form`, `dialog`, `dropdown-menu`.
    - Run: `npx shadcn@latest add ...`

## Deliverables
- A responsive shell with working navigation and correct branding colors.
